# Sales_Data_Analysis
Exploratory data analysis of a retail superstore sales dataset using Python, Pandas, Matplotlib and Seaborn to uncover revenue trends, profit insights and regional performance.
# Superstore Sales Analysis

## Project Overview
This project analyzes a retail superstore sales dataset to uncover 
patterns in revenue, profit, product performance, and regional trends 
using Python.

## Dataset
- **Source:** Kaggle - Superstore Sales Dataset
- **Size:** 9,994 rows, 21 columns
- **Period:** 2014 - 2017

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Research Questions
1. Which product category generates the most total sales?
2. Which month has the highest sales across all years?
3. Which region is the most profitable?
4. Which sub-category receives the highest average discount?
5. Are there products that sell a lot but generate little profit?
6. How has total sales changed year over year?
7. Which state has the highest number of orders?

## Key Findings
- Technology leads in total sales with $836,154
- November is the peak sales month due to holiday shopping
- The West region is the most profitable
- Binders receive the highest average discount
- Appliances and Binders are severely loss-making despite decent sales
- Sales show consistent year over year growth from 2014 to 2017
- California has the highest number of orders nationally

## How to Run
1. Clone the repository
2. Install required libraries: `pip install pandas matplotlib seaborn`
3. Open `Superstore_Sales_Analysis.ipynb` in Jupyter Notebook
4. Run all cells from top to bottom
